# 混合物MSDS GHS分类计算指南

> 数据来源：ChemSafetyPro、ECHA、OSHA、GBZ/T 230-2025、UN GHS紫皮书

---

## 1. 混合物GHS分类决策树

```
混合物
  ↓
有混合物整体实验数据？
  ├─ 是 → 直接使用实验数据分类
  └─ 否 → 能否使用桥接原则(Bridging Principles)？
           ├─ 是 → 使用桥接原则推断
           └─ 否 → 基于组分数据分类
                    ├─ 使用ATE加和公式（急性毒性等可加和危害）
                    └─ 使用切割值/浓度限值（非加和危害）
```

---

## 2. ATE加和公式（急性毒性计算）

### 核心公式

$$\frac{1}{ATE_{mix}} = \sum_{i=1}^{n} \frac{C_i}{ATE_i}$$

- **ATE_mix**: 混合物的急性毒性估计值 (mg/kg)
- **C_i**: 成分i的浓度（质量分数，如30%=0.30）
- **ATE_i**: 成分i的急性毒性估计值 (mg/kg)

### 未知毒性成分>10%时的修正公式

$$\frac{1}{ATE_{mix}} = \frac{\sum \frac{C_i}{ATE_i}}{100\% - C_{unknown}}$$

### ATE值获取方式

| 数据来源 | ATE值 |
|---------|-------|
| 已知LD50/LC50 | 直接使用实验值 |
| 仅有GHS分类 | 使用分类换算表 |

### GHS急性毒性分类换算表（口服）

| 分类 | LD50范围 (mg/kg) | 换算ATE值 (mg/kg) |
|------|-----------------|------------------|
| Cat 1 | ≤5 | 0.5 |
| Cat 2 | >5-50 | 5 |
| Cat 3 | >50-300 | 100 |
| Cat 4 | >300-2000 | 500 |
| Cat 5 | >2000-5000 | 2500 |

## 3. 计算示例（来自ChemSafetyPro）

### 示例1：已知LD50值

| 成分 | 浓度 | LD50 (mg/kg) |
|------|------|-------------|
| A | 30% | 500 |
| B | 20% | 200 |
| C（无毒） | 50% | - |

```
1/ATE = (0.30/500) + (0.20/200)
      = 0.0006 + 0.001
      = 0.0016

ATE = 625 mg/kg

→ 分类：急性口服毒性 Cat 4 (300-2000 mg/kg)，H302
```

### 示例2：仅知GHS分类

| 成分 | 浓度 | GHS分类 | 换算ATE |
|------|------|---------|---------|
| A | 10% | Cat 4 | 500 mg/kg |
| B | 5% | Cat 2 | 5 mg/kg |
| C（未知） | 85% | 无 | - |

```
1/ATE = (0.10/500) + (0.05/5)
      = 0.0002 + 0.01
      = 0.0102

ATE = 98 mg/kg

→ 分类：急性口服毒性 Cat 3 (50-300 mg/kg)，H301
```

## 4. 桥接原则（Bridging Principles）

| 原则 | 含义 | 适用条件 |
|------|------|---------|
| 稀释 | 已测试混合物A被较低危害成分B稀释 | A+B=A（分类不变） |
| 批次 | 同条件下生产的不同批次 | A=B |
| 高浓度混合物 | A已在最高危害类别分类，B浓度更高 | B=A |
| 插值 | 混合物A和B有相同危害成分和分类，C介于两者之间 | C=A=B |
| 实质相似 | A+B和C+B，A与C相似 | A+B=C+B |
| 气雾剂 | 非气雾形式已分类 | 气雾形式=非气雾形式 |

## 5. 切割值/浓度限值

### 5.1 可加和危害（使用加和原则）

- 急性毒性
- 皮肤腐蚀/刺激
- 严重眼损伤/眼刺激
- STOT-单次接触-cat3（呼吸道刺激/麻醉效应）
- 水生环境危害

### 5.2 非加和危害（不使用加和原则，使用浓度限值）

- 皮肤/呼吸致敏
- 生殖细胞致突变
- 致癌性
- 生殖毒性
- STOT-cat1和cat2
- 吸入危害

### 5.3 皮肤腐蚀/刺激浓度限值

| 组分分类 | 混合物分类触发浓度 |
|---------|------------------|
| 皮肤腐蚀 Cat 1A/1B/1C | ≥5% → Cat 1 |
| 皮肤腐蚀 Cat 1A/1B/1C | ≥1%但<5% → Cat 2 |
| 皮肤刺激 Cat 2 | ≥10% → Cat 2 |
| 皮肤刺激 Cat 2 | ≥1%但<10% → 考虑分类 |

### 5.4 严重眼损伤浓度限值

| 组分分类 | 混合物分类触发浓度 |
|---------|------------------|
| 严重眼损伤 Cat 1 | ≥3% → Cat 1 |
| 眼刺激 Cat 2A | ≥10% → Cat 2A |
| 眼刺激 Cat 2A | ≥1%但<10% → 考虑分类 |

## 6. 混合物中未知毒性成分的处理

- 当未知毒性成分**≤10%**：使用标准ATE公式
- 当未知毒性成分**>10%**：使用修正ATE公式
- 当未知毒性成分**≥1%**：需在SDS和标签中注明"x%的混合物由未知急性毒性的成分组成"
