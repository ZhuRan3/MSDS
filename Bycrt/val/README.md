## 验证用途

用于验证AI生成的MSDS与真实MSDS的数据一致性：
- CAS号、分子式、分子量
- GHS分类（H码、P码）
- 毒理学数据（LD50、LC50）
- 理化性质（闪点、沸点等）
- 运输信息（UN编号）
- 职业接触限值
