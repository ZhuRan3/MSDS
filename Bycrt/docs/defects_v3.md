# MSDS生成系统缺陷清单 V3

> 版本: 3.0 | 日期: 2026-05-03 | 来源: 6份最新MSDS全面审查（盐酸/硫酸/乙醇/丙酮 + 工业除漆剂/防锈处理液D型）
> 对照标准: GB/T 16483-2008、GB/T 17519-2013、UN GHS Rev.9、真实商用SDS

## 缺陷统计

| 优先级 | 数量 | 说明 |
|--------|------|------|
| P0 阻塞级 | 3 | 混合物分类完全失效、检索错配致命错误 |
| P1 高优先级 | 10 | 数据准确性、分类一致性、逻辑矛盾 |
| P2 中等优先级 | 5 | 格式规范、信息完整度 |

## P0 阻塞级缺陷

### NEW-101: 混合物GHS分类计算器返回0项——分类完全失效

- **严重性**: P0-致命
- **影响范围**: 所有混合物MSDS（工业除漆剂、防锈处理液等）
- **根因**: `mixture_calculator.py` 的 `build_component()` 构建组分时，`component.ghs_classifications` 可能为空列表，导致所有分类检查（腐蚀、易燃、毒性等）循环时累加器始终为0，无任何阈值被触发
- **涉及文件**:
  - `Bycrt/core/mixture_calculator.py` — `build_component()`, `check_skin_corrosion()`, `check_flammability()`
  - `Bycrt/core/sds_pipeline_v2.py` — 混合物流程调用
- **具体案例**:

| 混合物 | 应分类 | 实际输出 |
|--------|--------|----------|
| 防锈液(NaOH 5%+NaClO 15%+乙醇10%) | 皮肤腐蚀Cat1, 眼损伤Cat1, 易燃Cat2 | **GHS分类: 无可用数据** |
| 除漆剂(二氯甲烷60%+甲醇20%+甲苯20%) | 急性毒性Cat3, 致癌Cat2, 易燃Cat2, STOT反复Cat1 | **GHS分类: 无可用数据** |

- **后果**: S2整节空白→S4/S5/S7/S13全部降级为generic模板→S14运输分类矛盾→报告完全不可用
- **修复方案**: 在 `build_component()` 后加日志验证 `ghs_classifications` 是否被正确填充；确认 `load_component_from_kb()` 返回的DB字段名与 `build_component()` 读取的key完全一致

### NEW-102: "甲苯"被匹配为"三硝基甲苯"(TNT)——检索名称匹配严重错误

- **严重性**: P0-致命
- **影响范围**: 所有含甲苯的混合物
- **根因**: `evidence_fusion.py:243-257` 的 `_search_kb()` 使用 `query_lower in name_cn`（子串包含），"甲苯" 是 "三硝基甲苯" 的子串，且TNT(CAS 118-96-7)在 `chemical_db.json` 中排在甲苯(CAS 108-88-3)前面，循环先命中TNT即返回
- **涉及文件**: `Bycrt/core/evidence_fusion.py` — `_search_kb()` L243-257
- **具体错误**: 工业除漆剂S3成分表显示 "三硝基甲苯 | 118-96-7 | 20.0%"——完全错误的化学品
- **修复方案**:
  1. 先做精确匹配 (`query == name_cn`)，命中即返回
  2. 模糊匹配时按名称长度排序，短名优先（甲苯 < 三硝基甲苯）
  3. 同理修复 `_search_val_samples()` L260-272

### NEW-103: 混合物S14运输信息矛盾——有UN号却标注"不受管制"

- **严重性**: P0-致命（直接影响运输合规）
- **影响范围**: 所有混合物MSDS
- **根因**: 混合物S14运输块中，UN编号和运输类别来自首个组分的KB数据（如UN1823=NaOH），但分类结果0项时 `is_regulated` 判断为False，导致输出"不受危险货物规则限制"
- **涉及文件**: `Bycrt/core/sds_generator.py` — `generate_section_14()` 和混合物S14块
- **具体案例**:
  - 防锈液: UN编号=1823, 运输类别=8, 但正文写"不受危险货物规则限制"
  - 除漆剂: UN编号=1593, 运输类别=无可用数据, 但正文写"不受管制"
- **修复方案**: 当 `is_regulated` 为False但已有UN编号/运输类别时，应使用已有数据而非标注"不受管制"

## P1 高优先级缺陷

### NEW-104: S11眼损伤Cat2A描述使用Cat1严重模板——致灾描述等级不符

- **严重性**: P1-高
- **影响范围**: 乙醇、丙酮等Cat2A分类化学品
- **具体错误**: S11 "可能的主要症状" 写 "接触可引起严重的角膜损伤，可能造成不可逆的视力损害甚至失明"——这是Cat1（严重眼损伤）的描述。Cat2A（眼刺激）应为 "引起眼睛发红、疼痛、流泪，刺激持续可导致角膜损伤"
- **涉及文件**: `Bycrt/db/safety_knowledge.json` — eye_damage 描述模板
- **修复方案**: DEF-116已区分Cat1/2A/2B描述模板，但S11症状描述的查询逻辑可能未正确读取对应Cat等级

### NEW-105: S10酸类化学品自相矛盾——"与酸类接触可发生剧烈反应"

- **严重性**: P1-高
- **影响范围**: 盐酸、硫酸等酸类MSDS
- **具体错误**:
  - 盐酸S10: "危险反应的可能性: 与酸类接触可发生剧烈反应"——盐酸本身就是酸
  - 硫酸S10: "与碱类接触可发生反应"（正确），但随后又写 "遇水剧烈反应"——硫酸与水是稀释放热，非危险反应
- **涉及文件**: `Bycrt/core/sds_generator.py` — S10生成逻辑
- **修复方案**: 腐蚀性酸类型的S10应排除"与酸类接触"，改为"与碱类接触"。对硫酸的"遇水"描述改为"加水稀释时放热，应将酸缓慢加入水中"

### NEW-106: S10/S7水溶液化学品标注"避免潮湿环境和水"——对水溶液无意义

- **严重性**: P1-高
- **影响范围**: 盐酸(36%水溶液)、硫酸(水溶液)、NaOH溶液、防锈液等
- **具体错误**:
  - 盐酸S10: "应避免的条件: 潮湿环境、水"
  - 硫酸S10: "应避免的条件: 潮湿环境、水"
  - 防锈液S10: "应避免的条件: 潮湿环境、水"
- **涉及文件**: `Bycrt/db/safety_knowledge.json` — handling_storage 模板
- **修复方案**: 对已知的水溶液化学品（CAS在 `solution_concentrations` 映射中），S10"应避免的条件"应排除"潮湿环境、水"，替换为"高温"等

### NEW-107: S11生殖毒性标注"类别3"——GHS已无此分类等级

- **严重性**: P1-高
- **影响范围**: 乙醇、丙酮等有生殖毒性数据的化学品
- **具体错误**: S11 "生殖毒性: 类别3（可能对生育能力或胎儿有损害）"
- **根因**: `chemical_db.json` 中存储的是旧版分类标准，GHS Rev.7起生殖毒性只有Cat1A/1B/2
- **涉及文件**: `Bycrt/db/chemical_db.json` — 乙醇/丙酮 reproductive_toxicity 字段
- **修复方案**: 更新DB中的生殖毒性分类为GHS标准（Cat1A/1B/2），或标注"无分类"

### NEW-108: S9混合物外观"无色透明液体；白色固体"——固液矛盾

- **严重性**: P1-高
- **影响范围**: 含固体组分（NaOH等）的液体混合物
- **根因**: DEF-110修复后外观取主组分，但NaOH在DB中外观为"白色固体"，当NaOH非最大浓度组分时仍被拼接
- **具体案例**: 防锈液（水70%+NaOH 5%）：外观显示"无色透明液体；白色固体"
- **涉及文件**: `Bycrt/core/sds_generator.py` — 混合物S9外观聚合
- **修复方案**: 混合物含>50%液体组分时，外观应仅显示液体描述，不拼接固体组分纯品外观

### NEW-109: S1混合物英文名称/基本信息取首个组分——混合物信息缺失

- **严重性**: P1-高
- **影响范围**: 所有混合物MSDS
- **具体错误**:
  - 防锈液S1: 英文名称="sodium hydroxide"（NaOH的英文名，不是混合物名）
  - 除漆剂S1: 英文名称="不适用"
  - CAS号/分子式/分子量均显示"不适用"
- **涉及文件**: `Bycrt/core/sds_generator.py` — 混合物S1生成
- **修复方案**: 混合物S1英文名应标注" mixture"后缀或留空；CAS号标注"混合物——见S3成分表"

### NEW-110: 硫酸分子式"H2O4S"——非标准化学式写法

- **严重性**: P1-高
- **影响范围**: 硫酸MSDS
- **具体错误**: S1/S3分子式显示"H2O4S"（PubChem标准格式），但商用MSDS应显示"H₂SO₄"
- **涉及文件**: `Bycrt/db/chemical_db.json` — 硫酸 molecular_formula 字段
- **修复方案**: 将硫酸分子式从"H2O4S"改为"H2SO4"

### NEW-111: S9混合物易燃性标注"不适用"——含易燃组分时矛盾

- **严重性**: P1-高
- **影响范围**: 含易燃组分的混合物（防锈液含10%乙醇、除漆剂含20%甲醇）
- **具体错误**: 防锈液S9易燃性="不适用"，但S3显示乙醇(Cat2)占10%；除漆剂S9易燃性="不适用"，但含20%甲醇
- **涉及文件**: `Bycrt/core/sds_generator.py` — 混合物S9易燃性生成
- **修复方案**: 当混合物含易燃液体组分且总浓度≥10%时，S9易燃性应标注"易燃"或"含易燃组分"

### NEW-112: S8混合物OEL多组分重复无标识——限值表不可读

- **严重性**: P1-高
- **影响范围**: 所有含多组分的混合物MSDS
- **具体错误**: 防锈液S8 OEL表显示两组重复的"PC-TWA/PC-STEL/ACGIH TLV-TWA"但无化学品名称区分：
  ```
  PC-TWA | 2 mg/m³       (NaOH的)
  PC-TWA | 未设定         (乙醇的)
  ```
- **涉及文件**: `Bycrt/core/sds_generator.py` — 混合物S8 OEL生成
- **修复方案**: 多组分OEL应标注化学品名称（如"PC-TWA[NaOH]"→"PC-TWA（氢氧化钠）"）

### NEW-113: S13混合物废弃处置过于笼统——缺少危废类别

- **严重性**: P1-高
- **影响范围**: 所有混合物MSDS
- **具体错误**: 防锈液/除漆剂S13 "处置方法: 按照国家和地方相关法规要求进行处置"——未标注具体危废类别
- **涉及文件**: `Bycrt/core/sds_generator.py` — 混合物S13生成
- **修复方案**: 根据组分自动推导危废类别（含有机溶剂→HW06、含酸→HW34、含碱→HW35）

## P2 中等优先级缺陷

### NEW-114: S1推荐用途/供应商/应急电话全部"待确认"

- **严重性**: P2-中
- **影响范围**: 所有MSDS
- **根因**: 这些字段需要用户输入，系统无默认值
- **修复方案**: S1应有参数化接口接收企业信息；默认值可改为"工业用途"等通用描述

### NEW-115: S9 pH值全部"待确认"

- **严重性**: P2-中
- **影响范围**: 所有MSDS
- **修复方案**: 对已知酸碱化学品自动推导pH范围（盐酸~0-1、硫酸~0-1、NaOH 1%溶液~13.5、乙醇~7）

### NEW-116: S11盐酸LD50=900 mg/kg缺少实验动物信息

- **严重性**: P2-中
- **影响范围**: 盐酸等部分化学品
- **具体错误**: "LD50经口: 900 mg/kg"——缺少"(大鼠)"等实验动物信息
- **修复方案**: KB数据中LD50应包含实验动物信息

### NEW-117: S15硫酸/盐酸缺少《易制毒化学品管理条例》

- **严重性**: P2-中
- **影响范围**: 盐酸、硫酸MSDS
- **具体错误**: S15法规列表缺少易制毒法规——盐酸/硫酸均为第三类易制毒化学品
- **涉及文件**: `Bycrt/db/chemical_db.json` — CAS 7647-01-0/7664-93-9 缺少"易制毒"标记
- **修复方案**: 在 `chemical_db.json` 或 `chemical_type_rules.json` 中添加易制毒CAS列表，S15按CAS匹配

### NEW-118: S11皮肤刺激与S2 GHS分类不一致

- **严重性**: P2-中
- **影响范围**: 乙醇、丙酮等
- **具体错误**: S11 "皮肤腐蚀/刺激: 类别2（皮肤刺激）"，但S2 GHS分类中未列出皮肤刺激
- **修复方案**: S11应仅显示S2 GHS分类中实际列出的危害，或S2应补全该分类

## 与真实商用SDS的关键差距总结

| 差距领域 | 真实商用SDS | 本系统当前状态 | 严重性 |
|----------|------------|---------------|--------|
| 混合物GHS分类 | 基于GHS加和/桥接原则完整分类 | **分类完全失效(0项)** | P0 |
| 检索准确性 | 精确匹配化学品 | 子串匹配导致甲苯→TNT | P0 |
| 运输一致性 | UN号+类别+描述统一 | 有UN号却标注"不受管制" | P0 |
| 眼损伤描述 | 按Cat等级精确描述 | Cat2A用Cat1描述 | P1 |
| 逻辑自洽性 | 酸不写"避免与酸接触" | 盐酸/硫酸S10自相矛盾 | P1 |
| 分子式格式 | 标准化学式(H₂SO₄) | PubChem格式(H2O4S) | P1 |
| 企业标识 | 供应商/用途/应急电话完整 | 全部"待确认" | P2 |
| pH值 | 有具体值或范围 | 全部"待确认" | P2 |
| 危废分类 | 按组分标注HW类别 | 混合物废弃处置笼统 | P1 |
