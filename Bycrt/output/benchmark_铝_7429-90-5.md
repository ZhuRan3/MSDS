# 第四部分：急救措施

**吸入**: 将患者移至空气新鲜处。如呼吸困难，给予吸氧。如呼吸停止，进行人工呼吸。立即就医。
注意：铝粉尘可能引起呼吸道刺激。 <!-- src:kb conf:1.0 -->

**皮肤接触**: 用大量水和肥皂冲洗。如有刺激，就医。

**眼睛接触**: 用大量水冲洗几分钟。如持续刺激，就医。

**食入**: 漱口。给水稀释。如感觉不适，就医。

# 第五部分：消防措施

**危险特性**: 遇明火、高热可燃。
金属粉末在空气中可燃。与水或潮湿空气反应产生氢气，氢气易燃。与酸碱反应产生氢气。不可用水灭火。 <!-- src:kb conf:1.0 -->

**适用灭火剂**: 干粉、二氧化碳、泡沫、砂土。

**禁止使用的灭火剂**: 无特殊禁止要求。

**消防建议**: 消防人员必须佩戴防毒面具，穿全身消防服。尽可能将容器从火场移至空旷处。喷水保持火场容器冷却。
