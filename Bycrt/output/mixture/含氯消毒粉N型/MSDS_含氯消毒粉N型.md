# 第四部分：急救措施

**吸入**: 立即将患者移至空气新鲜处。如呼吸困难，给予吸氧。如呼吸停止，进行人工呼吸（注意：施救者勿口对口接触腐蚀性气体）。立即就医。

- [三氯异氰尿酸] 迅速脱离现场至空气新鲜处。保持呼吸道通畅。如呼吸困难，给输氧。就医。
- [碳酸钠] 迅速脱离现场至空气新鲜处。如呼吸困难，给输氧。就医。
- [十二烷基苯磺酸] 迅速脱离现场至空气新鲜处。如呼吸困难，给输氧。就医。
- 可能引起呼吸道刺激或肺水肿。延迟观察至少48小时。

**皮肤接触**: 立即脱去被污染的衣物。用大量流动清水冲洗至少30分钟。不可使用化学中和剂。冲洗时注意不要将化学品扩散到未受影响的皮肤。立即就医。

- [三氯异氰尿酸] 立即脱去污染的衣着，用大量流动清水冲洗至少15分钟。就医。
- [碳酸钠] 脱去污染的衣着，用大量流动清水冲洗。
- [十二烷基苯磺酸] 立即脱去污染的衣着，用大量流动清水冲洗至少15分钟。就医。

**眼睛接触**: 立即翻开上下眼睑，用大量流动清水或生理盐水冲洗至少30分钟。如戴隐形眼镜且易取下，先取下再继续冲洗。不可揉眼睛。立即送往眼科就医。

- [三氯异氰尿酸] 立即提起眼睑，用大量流动清水或生理盐水彻底冲洗至少15分钟。就医。
- [碳酸钠] 立即提起眼睑，用大量流动清水或生理盐水冲洗至少15分钟。就医。
- [十二烷基苯磺酸] 立即提起眼睑，用大量流动清水或生理盐水彻底冲洗至少15分钟。立即就医。

**食入**: 不要催吐（腐蚀性物质催吐可造成食道二次损伤）。不要经口给任何物质。立即就医。如患者意识清醒，可用水漱口。

- [三氯异氰尿酸] 用水漱口，给饮牛奶或蛋清。禁止催吐。立即就医。
- [碳酸钠] 用水漱口，饮用大量水。禁止催吐。就医。
- [十二烷基苯磺酸] 用水漱口。禁止催吐。饮用牛奶或蛋清保护胃黏膜。立即就医。


**急性中毒症状**: 吸入粉尘可引起呼吸道刺激、咳嗽、呼吸困难；皮肤接触可引起刺激；眼睛接触可引起严重刺激；误食可引起口腔、胃肠道刺激。

# 第五部分：消防措施

**危险特性**: 强氧化剂。与可燃物接触可能引起燃烧。受热分解放出氧气，可加剧燃烧。容器在火中可能发生爆炸。 燃烧产生有毒气体。火灾产生的有毒燃烧产物可能危害消防人员和周围人群。 与金属反应可产生易燃的氢气。受热或遇水分解放出腐蚀性气体。容器在火中可能破裂。
- [三氯异氰尿酸] 氧化剂，可加剧燃烧。发生火灾时可用大量水灭火。禁止使用干化学品或泡沫灭火剂。消防人员须穿戴全身防护服和正压自给式呼吸器。
- [碳酸钠] 不燃。使用适合周围火源的灭火剂。
- [十二烷基苯磺酸] 使用水雾、泡沫、干粉或二氧化碳灭火。消防人员须穿戴全身防护服和正压自给式呼吸器。
- 燃烧可能产生剧毒氰化氢(HCN)气体。
- 燃烧可能产生有毒氯气。
- 燃烧可能产生有毒氮氧化物。
- 燃烧可能产生有毒二氧化硫。

**适用灭火剂**: 大量水、二氧化碳、泡沫、水雾、干砂。

**禁止使用的灭火剂**: 禁止使用干粉、泡沫（有机物成分可能与氧化剂反应）。

**消防建议**: 消防人员应佩戴自给式正压呼吸器，穿全身化学防护服。不要让氧化剂与可燃物接触。大量用水冷却容器。

**燃烧/分解产物**: 受热分解放出有毒的氯气、氮氧化物、一氧化碳、氰化氢；燃烧时产生硫氧化物、一氧化碳、二氧化碳；高温分解生成氧化钠和二氧化碳（推断值）。含氯有机物不完全燃烧可产生光气（碳酰氯）和氯化氢等剧毒气体

# 第六部分：泄漏应急处理

**人员防护措施**: 佩戴防护手套、护目镜。避免吸入蒸气。穿戴防酸碱防护服。

**环境保护措施**: 防止进入排水沟、下水道、水体和土壤。如泄漏物进入水体，立即通知相关部门。

**收容与清除方法**: 收集泄漏物。回收利用。避免产生粉尘。用石灰石、苏打灰或石灰中和酸液后再行清理。

# 第七部分：操作处置与储存

**操作注意事项**: 密闭操作，注意通风。操作尽可能机械化、自动化。操作人员必须经过专门培训，严格遵守操作规程。建议操作人员佩戴头罩型电动送风过滤式防尘呼吸器，穿橡胶耐酸碱服，戴橡胶耐酸碱手套。远离易燃、可燃物。避免产生酸雾。防止蒸气释放到工作场所空气中。避免与碱类、胺类、碱金属接触。搬运时要轻装轻卸，防止包装及容器损坏。配备泄漏应急处理设备。倒空的容器可能残留有害物。

**储存条件**: 储存于阴凉、通风的库房。库温不超过30°C，相对湿度不超过85%。保持容器密封。应与碱类、胺类、碱金属、易燃物、可燃物分开存放，切忌混储。储区应备有泄漏应急处理设备和合适的收容材料。耐酸地坪。

**禁配物**: 二氧化硫、可燃物、含氮化合物、强氧化剂、强碱、强酸、氟、碱金属、磷、胺类、还原剂、重金属盐、金属、铝、铵盐、锂
