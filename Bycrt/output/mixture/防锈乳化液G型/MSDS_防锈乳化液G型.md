# 第四部分：急救措施

**吸入**: 将患者移至空气新鲜处。如呼吸困难，给予吸氧。如呼吸停止，进行人工呼吸。如持续不适，就医。

- [矿物油（白油）] 迅速脱离现场至空气新鲜处。如呼吸困难，给输氧。就医。
- [油酸] 迅速脱离现场至空气新鲜处。如呼吸困难，给输氧。就医。
- [三乙醇胺] 迅速脱离现场至空气新鲜处。如呼吸困难，给输氧。就医。
- [硼酸] 迅速脱离现场至空气新鲜处。如呼吸困难，给输氧。就医。
- [苯并三氮唑] 迅速脱离现场至空气新鲜处。如呼吸困难，给输氧。就医。

**皮肤接触**: 用大量水和肥皂冲洗。如有刺激，就医。

- [矿物油（白油）] 用肥皂和水清洗皮肤。
- [油酸] 脱去污染的衣着，用大量流动清水冲洗。
- [三乙醇胺] 脱去污染的衣着，用大量流动清水冲洗。
- [硼酸] 脱去污染的衣着，用大量流动清水冲洗。
- [苯并三氮唑] 立即脱去污染的衣着，用大量流动清水冲洗。

**眼睛接触**: 用大量水冲洗几分钟。如持续刺激，就医。

- [矿物油（白油）] 提起眼睑，用流动清水冲洗。如持续不适，就医。
- [油酸] 提起眼睑，用流动清水或生理盐水冲洗。如持续不适，就医。
- [三乙醇胺] 提起眼睑，用流动清水或生理盐水冲洗至少15分钟。如持续不适，就医。
- [硼酸] 提起眼睑，用流动清水或生理盐水冲洗。如持续不适，就医。
- [苯并三氮唑] 立即提起眼睑，用大量流动清水或生理盐水彻底冲洗至少15分钟。立即就医。

**食入**: 漱口。给水稀释。如感觉不适，就医。

- [矿物油（白油）] 禁止催吐（防止误吸入肺）。立即就医。
- [油酸] 用水漱口。如感觉不适，就医。
- [三乙醇胺] 用水漱口，饮用大量水。禁止催吐。就医。
- [硼酸] 用水漱口，饮用大量水。禁止催吐。立即就医。
- [苯并三氮唑] 用水漱口。禁止催吐。立即就医。


**救援人员防护**: 救援人员应佩戴适当的防护装备。避免直接接触。
