# 第四部分：急救措施

**吸入**: 将患者移至空气新鲜处。保持安静、保暖。如呼吸困难，给予吸氧。如呼吸停止，进行人工呼吸（注意：施救者勿吸入蒸气）。如出现中毒症状就医。

- 可能导致头晕、嗜睡。保持患者清醒。

**皮肤接触**: 立即脱去被污染的衣物。用大量肥皂水和清水彻底冲洗皮肤。被污染的衣物须彻底清洗后方可重新使用。如刺激持续，就医。

**眼睛接触**: 立即用大量清水冲洗几分钟。如戴隐形眼镜且易取下，取下后继续冲洗。如持续刺激或疼痛，就医。

**食入**: 不要催吐（有吸入性肺炎风险）。如患者意识清醒，用水漱口。给200-300mL水稀释（在意识清醒情况下）。禁止给失去意识者经口任何物质。立即就医。

- [丙酮] 不要催吐
- [乙醇] 不要催吐


**救援人员防护**: 救援人员应佩戴适当的防护装备。注意远离火源。避免吸入蒸气。

# 第五部分：消防措施

**危险特性**: 极易燃。其蒸气与空气可形成爆炸性混合物。蒸气比空气重，能沿地面扩散至远处，遇火源可能引起回燃。容器在火中可能破裂或爆炸。
- [丙酮] 极易燃。其蒸气与空气混合形成爆炸性混合物。蒸气能扩散到火源处并回燃。容器在火中可能破裂。
- [乙醇] 极易燃。其蒸气与空气混合形成爆炸性混合物。

**适用灭火剂**: 干粉、二氧化碳、泡沫、水雾。

**禁止使用的灭火剂**: 禁止使用直流水（可能扩大火势和化学品扩散范围）。
