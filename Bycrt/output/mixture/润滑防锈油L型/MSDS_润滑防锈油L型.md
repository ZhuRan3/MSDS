# 第四部分：急救措施

**吸入**: 立即将患者移至空气新鲜处。保持安静、保暖。如呼吸困难，给予吸氧。如呼吸停止，进行人工呼吸（注意：避免口对口人工呼吸，应使用器械通气）。立即就医。

- [矿物油（白油）] 迅速脱离现场至空气新鲜处。如呼吸困难，给输氧。就医。
- [油酸] 迅速脱离现场至空气新鲜处。如呼吸困难，给输氧。就医。
- [三乙醇胺] 迅速脱离现场至空气新鲜处。如呼吸困难，给输氧。就医。
- [苯并三氮唑] 迅速脱离现场至空气新鲜处。如呼吸困难，给输氧。就医。

**皮肤接触**: 立即脱去被污染的衣物。用大量肥皂水和清水冲洗皮肤。被污染的衣物须彻底清洗后方可重新使用。立即就医。

- [矿物油（白油）] 用肥皂和水清洗皮肤。
- [油酸] 脱去污染的衣着，用大量流动清水冲洗。
- [三乙醇胺] 脱去污染的衣着，用大量流动清水冲洗。
- [苯并三氮唑] 立即脱去污染的衣着，用大量流动清水冲洗。

**眼睛接触**: 立即用大量清水冲洗至少15分钟，提起上下眼睑确保彻底冲洗。如戴隐形眼镜且易取下，先取下再继续冲洗。立即就医。

- [矿物油（白油）] 提起眼睑，用流动清水冲洗。如持续不适，就医。
- [油酸] 提起眼睑，用流动清水或生理盐水冲洗。如持续不适，就医。
- [三乙醇胺] 提起眼睑，用流动清水或生理盐水冲洗至少15分钟。如持续不适，就医。
- [苯并三氮唑] 立即提起眼睑，用大量流动清水或生理盐水彻底冲洗至少15分钟。立即就医。

**食入**: 不要催吐。如患者意识清醒，用水漱口。禁止给失去意识者经口任何物质。立即呼叫中毒控制中心或就医。

- [矿物油（白油）] 禁止催吐（防止误吸入肺）。立即就医。
- [油酸] 用水漱口。如感觉不适，就医。
- [三乙醇胺] 用水漱口，饮用大量水。禁止催吐。就医。
- [苯并三氮唑] 用水漱口。禁止催吐。立即就医。


**急性中毒症状**: 误食可引起腹泻；吸入油雾可引起呼吸道刺激。

**救援人员防护**: 救援人员必须佩戴适当的呼吸防护和化学防护装备。避免直接接触。确保自身安全后再进行施救。

**对医生的提示**: 本品的毒性作用可能延迟出现。至少观察24-48小时。根据暴露途径和剂量进行对症治疗。

# 第五部分：消防措施

**危险特性**: 燃烧产生有毒气体。火灾产生的有毒燃烧产物可能危害消防人员和周围人群。
- [矿物油（白油）] 使用水雾、泡沫、干粉或二氧化碳灭火。消防人员须穿戴全身防护服。
- [油酸] 使用水雾、泡沫、干粉或二氧化碳灭火。消防人员须穿戴全身防护服。
- [三乙醇胺] 使用水雾、泡沫、干粉或二氧化碳灭火。消防人员须穿戴全身防护服。
- [苯并三氮唑] 使用水、泡沫、干粉或二氧化碳灭火。消防人员须穿戴全身防护服和正压自给式呼吸器。
- 燃烧可能产生有毒氮氧化物。

**适用灭火剂**: 干粉、二氧化碳、泡沫、水雾。

**禁止使用的灭火剂**: 禁止使用直流水（可能扩大有毒物质扩散范围）。

**消防建议**: 消防人员必须佩戴自给式正压呼吸器，穿全身化学防护服。从上风方向灭火。防止灭火废水流入水体。火灾现场应设置警戒线。

**燃烧/分解产物**: 燃烧时产生刺激性烟雾、一氧化碳、二氧化碳；燃烧时产生氮氧化物、一氧化碳、二氧化碳；高温分解产生一氧化碳、二氧化碳、烟雾（推断值）
